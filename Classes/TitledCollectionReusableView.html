TitledCollectionReusableView

A TitledCollectionReusableView is a UICollectionReusableView with a single UILabel intended for use as an analog to a UITableView header title (via tableView(_:titleForHeaderInSection:)) when using a CollectionViewFetchedResultsDelegateProvider. These views can be created via a TitledCollectionReusableViewFactory.
